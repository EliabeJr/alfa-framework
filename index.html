Features

Easy to use

Build web pages with few commands. Great for beginners, Alpha is a great way to learn SASS, NodeJS and new ways to program javascript, like PugJS.

Customizable

The variables are centralized in a single file, allowing faster changes. We cherish by code reuse, we comment all the functions for a good learning curve.

Why Alpha?

Every time that we start a new project, we have to decide which framework to use or begin a new stylesheet, sometimes the simple it's the best. Alpha it's a simple framework CSS to build big things.

We love the simple things.
